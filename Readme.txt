How to run the Ujianin Project
1. Download the  zip file
2. Extract the file and copy ujianin folder
3. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name pegawai_data
6. Import student_data.sql file(given inside the zip package in database file folder)
7.Run the script http://localhost/ujianin (frontend)
8. For admin panel http://localhost/ujianin/admin  (admin panel)
